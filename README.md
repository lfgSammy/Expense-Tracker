# Expense Tracker API

A simple Expense Tracker API built with Django REST Framework and JWT Authentication.

This API allows users to:
- Register and log in
- Create and manage expenses
- Track spending
- Filter expenses by week or month
- Update and delete expenses securely

---

## Features

- User Authentication with JWT
- Create Expense
- View All Expenses
- View Single Expense
- Update Expense
- Delete Expense
- Weekly Expense Filtering
- Monthly Expense Filtering
- Protected Routes
- User-specific data access

---

## Tech Stack

- Python
- Django
- Django REST Framework
- Simple JWT
- SQLite
